# 🐍 SnakeName 贪吃蛇大作战

一款基于 HTML5 Canvas 开发的网页版贪吃蛇游戏，界面精美、交互流畅，支持多难度选择与本地最高分记录。

## ✨ 功能特性

- **三档难度可选**：简单、普通、困难三种速度模式，满足不同游玩需求
- **最高分记录**：自动保存本地历史最高分，支持持久化存储，挑战自我极限
- **完整游戏流程**：包含开始菜单、游戏暂停、结束重开、返回主菜单全流程交互
- **精美视觉效果**：渐变背景、毛玻璃面板、圆角元素、食物发光特效、蛇身渐变透明度
- **多套操控方式**：同时支持方向键与 WASD 按键，操作顺手自然
- **即开即玩**：纯前端单文件实现，无需安装任何依赖，浏览器直接运行

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / W A S D | 控制蛇的移动方向 |
| P / p | 暂停 / 继续游戏 |
| 空格键 | 游戏结束后快速重新开始 |


## 💻 本地运行

1. 下载仓库中的 `index.html` 文件到本地
2. 直接双击文件，使用任意现代浏览器打开即可开始游戏

## 🛠️ 技术栈

- **HTML5**：页面结构与 Canvas 画布渲染
- **CSS3**：界面样式、渐变动画、毛玻璃特效
- **原生 JavaScript**：游戏核心逻辑、碰撞检测、状态管理
- **LocalStorage**：本地最高分数据持久化存储
